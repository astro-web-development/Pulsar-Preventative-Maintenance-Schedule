# Pulsar Preventative Maintenance Scheduler
> **Industrial-Strength Preventative Maintenance Engine**

Pulsar Preventative is an intuitive, single-page web application designed for manufacturing plants, machine shops, and facility management teams. It features an ISO 9001-compliant live print preview panel, flexible recurrence dispatch modeling, localized structural caching, and advanced administrative lockouts.

---

## 🚀 Key Features

*   **Dynamic Matrix Dashboard:** Real-time visibility into active PM profiles, critical component warnings, and total maintenance throughput metrics.
*   **Production Calendar Hub:** Full-scale interactive calendar tracking. Supports immediate modification of root records via traditional click drop mechanics or instant dispatching of recurring operational sheets directly from automated timeline nodes.
*   **Interactive Task Protocol Builder:** Define exact procedural engineering tasks, adjust execution intervals, map technicians, and capture explicit condition metrics.
*   **ISO 9001 Compliant Live-Preview:** Instantly renders inputs down into standard professional physical documentation, ready for single-click PDF export.
*   **Historical Document Vault:** Seamlessly transitions completed workspace files away from active logs into a centralized permanent historical repository.
*   **Administrative Terminal Security:** Features a robust 4-digit PIN lock security layer to restrict unauthorized views, alongside instant JSON database migration backup engines.

---

## 🛠️ Architecture & Tech Stack

This software is built entirely as an agile client-side stack requiring zero external backend installations or server configurations.

*   **Core UI Layout:** HTML5, Tailwind CSS CDN (Modern component framing)
*   **Visual Elements:** FontAwesome v6.4.0 (Operational iconography mapping)
*   **Data Timelines:** FullCalendar.js v6.1.8 (Industrial scheduling logic)
*   **Document Generation:** html2pdf.js v0.10.1 (Direct client-side DOM-to-PDF rendering)
*   **Persistence Layer:** Web Browser `localStorage` API
